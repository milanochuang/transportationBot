## 如何取得Token
* 按照您的需求，勾選於OAuth2的核選方塊後...
* 複製在核選方塊下方的認證網址（URL）並貼入瀏覽器執行
* 到Bot/BUILD-A-BOT，並複製下方的TOKEN
* 將Token貼入[thsr.bot](https://github.com/milanochuang/transportationBot/tree/master/Discord)中的```DISCORD_TOKEN=""```再執行，你的機器人就上線囉！
